Guide

Pour accéder aux ressources et les modifier, vous pouvez utiliser les méthodes HTTP :

GET POST PUT PATCH DELETE OPTIONS

Les exemples suivants utilisent l'API Fetch. Vous êtes libres d'utiliser une autre solution (Axios, Ky…).

Récupérer toutes les données


          fetch('https://o-contact-server.vercel.app/api/contacts')
            .then((response) => response.json())
            .then((json) => console.log(json))

          // Output
          [
            { id: 1, firstName: 'Mummy' /* ... */ },
            /* ... */
            { id: 6, firstName: 'Bob' /* ... */ }
          ]

Récupérer une donnée


          fetch('https://o-contact-server.vercel.app/api/contacts/1')
            .then((response) => response.json())
            .then((json) => console.log(json))

          // Output
          {
            id: 1,
            firstName: 'Mummy',
            lastName: 'Lopper',
            phone: '06 03 12 19 79',
            email: 'mummy@pyramid.com',
            category: 'family',
            favorite: false
          }

Ajouter une donnée


          fetch('https://o-contact-server.vercel.app/api/contacts', {
            method: 'POST',
            body: JSON.stringify({
              firstName: 'Dwight',
              lastName: 'Schrute',
              phone: '308-826-5962',
              email: 'beetfarmer97@dundermifflin.com',
              category: 'work',
              favorite: false
            }),
            headers: {
              'Content-type': 'application/json; charset=UTF-8'
            }
          })
            .then((response) => response.json())
            .then((json) => console.log(json))

          // Output
          {
            id: 7,
            firstName: 'Dwight',
            lastName: 'Schrute',
            phone: '308-826-5962',
            email: 'beetfarmer97@dundermifflin.com',
            category: 'work',
            favorite: false
          }

Important : la donnée ne sera pas vraiment créée sur le serveur, mais c'est comme si… En d'autres termes, si vous essayez d'accéder à la donnée avec l'id 7, vous aurez une erreur 404.

Mettre à jour une donnée

Méthode PUT


          fetch('https://o-contact-server.vercel.app/api/contacts/4', {
            method: 'PUT',
            body: JSON.stringify({
              id: 4,
              firstName: 'Henriette',
              lastName: 'Vonement',
              phone: '06 89 67 23 45',
              email: 'henri.vonement@pm.me',
              category: 'friends',
              favorite: false
            }),
            headers: {
              'Content-type': 'application/json; charset=UTF-8'
            }
          })
          .then((response) => response.json())
          .then((json) => console.log(json))

        // Output
        {
          id: 4,
          firstName: 'Henriette',
          lastName: 'Vonement',
          phone: '06 89 67 23 45',
          email: 'henri.vonement@pm.me',
          category: 'friends',
          favorite: false
        }

Méthode PATCH


          fetch('https://o-contact-server.vercel.app/api/contacts/4', {
            method: 'PATCH',
            body: JSON.stringify({
              firstName: 'Henriette'
            }),
            headers: {
              'Content-type': 'application/json; charset=UTF-8'
            }
          })
          .then((response) => response.json())
          .then((json) => console.log(json))

        // Output
        {
          id: 4,
          firstName: 'Henriette',
          lastName: 'Vonement',
          phone: '06 89 67 23 45',
          email: 'henri.vonement@pm.me',
          category: 'friends',
          favorite: false
        }

Important : la donnée ne sera pas vraiment mise à jour sur le serveur, mais c'est comme si…

Supprimer une donnée


          fetch('https://o-contact-server.vercel.app/api/contacts/6', {
            method: 'DELETE'
          })

Important : la donnée ne sera pas vraiment supprimer du serveur, mais c'est comme si…

Filtrer les données

Un filtrage de base est pris en charge par les paramètres de la requête.


          // Retourne tous les contacts marqués comme favoris
          fetch('https://o-contact-server.vercel.app/api/contacts?favorite=true')
            .then((response) => response.json())
            .then((json) => console.log(json))
